# Bare-Metal "Hello World" on RISC-V

Minimal bare-metal RISC-V UART project running on QEMU.
Implements UART output from scratch using RISC-V assembly, linker scripts, and memory-mapped I/O without an OS.

Supporting articles: 
- Part 1: https://akshatsharma05.github.io/posts/riscv-1/
- Part 2: https://akshatsharma05.github.io/posts/riscv-1/
